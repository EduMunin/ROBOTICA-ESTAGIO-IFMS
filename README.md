Projeto de Robótica - IFMS
Desenvolvimento de projetos de robótica utilizando kit LEGO e arduino para competições de robótica 
representando o IFMS, e desenvolvimento com arduino para apresentação escolar 
em prol de convidar e despertar o interesse de alunos do ensino fundamental pela tecnologia da informação.

Sobre o Projeto
Este repositório centraliza os códigos e documentações dos projetos de robótica desenvolvidos no campus.

Tecnologias e Ferramentas
Hardware: Arduino e Lego
Linguagens: C/C++
